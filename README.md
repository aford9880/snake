# Snake Game

Красивая змейка с неоновым оформлением,粒子 эффектами и встроенным искусственным интеллектом.

## Features

- 🎮 Управление: стрелки / WASD
- 🤖 Встроенный ИИ с многошаговым планированием
- 🎨 Неоновый градиентный фон с анимированной сеткой
- ✨ Частицы при поедании еды
- ⭐ Бонусная еда каждые 50 очков
- ⚡ Слайдер скорости (3–500ms)
- 📱 Touch/swipe на мобильных
- 🏆 Рекорд сохраняется в localStorage

## Launch

Открой `index.html` в любом браузере.

## Controls

| Key | Action |
|-----|--------|
| ← ↑ → ↓ | Движение |
| W A S D | Движение |
| ✨ Авто | Вкл/выкл ИИ |
| Слайдер слева | Скорость игры |

## AI

ИИ использует BFS-поиск пути к еде, оценку пространства (flood fill),
многошаговую симуляцию (8 ходов вперёд) и автоматический сброс цикла
случайным шумом при застревании.
